## <a name="interface-overview"></a>📱 Interface Overview

DocuMind features a clean, intuitive interface:
- **Home Screen:** Manage your notebooks and recent documents.
- **Reader View:** View extracted Markdown content with AI-generated summaries.
- **Chat Interface:** Interact with your document using natural language.

---

## <a name="notebooks"></a>📚 Notebooks

Organize your research by creating Notebooks. Each Notebook can contain multiple documents related to a single topic.

---

## <a name="summarization"></a>✍️ Summarization

When you upload a document, the system automatically generates:
1. **Quick Summary:** A paragraph summarizing the main idea.
2. **Key Takeaways:** Bullet points of the most critical information.

---

## <a name="chat"></a>💬 AI Chat

Ask questions about your documents in the chat tab. The AI uses the context of your notebooks to provide accurate, cited answers.
